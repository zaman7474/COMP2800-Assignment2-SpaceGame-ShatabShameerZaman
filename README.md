# Space Invaders — COMP2800 Assignment 2

A Space Invaders-style web game built from scratch using JavaScript and the HTML5 Canvas API, developed in GitHub Codespaces and deployed via GitHub Pages.

## Live Demo
https://zaman7474.github.io/COMP2800-Assignment2-SpaceGame-ShatabShameerZaman/

## How to Run
Open the live link in any modern browser. No installation required.

## Controls
| Key | Action |
|---|---|
| W / ↑ | Move Up |
| S / ↓ | Move Down |
| A / ← | Move Left |
| D / → | Move Right |
| Space | Shoot |
| R | Return to Menu |

## Levels
- **Level 1** — Enemy grid, 8 columns, 12 rows revealed in waves
- **Level 2** — Enemy grid + homing asteroids
- **Level 3** — Survive 3 minutes of asteroid shower

## Scoring
- 10 points per enemy destroyed
- Complete under 5 minutes → score x10
- Final score multiplied by remaining health

## Features
- Game menu with Play and Settings
- Music and sound effects with on/off toggles
- Health system with score multiplier
- Homing asteroids targeting player coordinates
- Enemy shooting (bottom 4 rows only)
- Explosion animations via canvas math
- Best score and previous score tracked per session
- Deployed on GitHub Pages

## Technologies
- HTML5 Canvas API
- Vanilla JavaScript
- GitHub Codespaces
- GitHub Pages
